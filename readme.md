# Welcomet to the video streaming app

## This is a simple video streaming app that allows you to upload and watch videos.

### Features:
- Upload videos
- Watch videos

### Technologies used:
- [FFmpeg](https://ffmpeg.org/download.html)
- Node.js
- Express.js
- React.js
